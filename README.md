# PasteBin
一个剪贴板的后端  
写着玩的 XD

## 环境
- Node.JS ≥ 18
- Redis ≥ 7.0
- pnpm

## 配置
将 `.env.example` 重命名为 `.env`，修改之后保存  
前端请见 https://github.com/BLxcwg666/PasteBin

## 运行
克隆本仓库  
运行`pnpm i`和`pnpm start`
